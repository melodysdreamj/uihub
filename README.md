## Usage

1. To add UI to your project, enter the following command in the terminal at the root of your flutter project:
   ```bash
   npm i -g uihub-cli
   uihub get NewUI
   ```
2. run the following command to run the UI: 
    ```bash
    flutter run -d chrome lib/uihub/NewUI/main.dart
    ```
